Gamma and Beta Function Solved Examples

Gamma Function

Gamma Function Formula: \[ \Gamma(n)=\int_0^{\infty} e^{-x}x^{n-1}dx \] \[ \Gamma\left(\frac12\right)=\sqrt{\pi} \]

Example 1

Evaluate:

\[ I=\int_0^{\infty} x^4 e^{-x^2}dx \]

Solution

\[ x^2=t \] \[ 2x\,dx=dt \] \[ dx=\frac{dt}{2\sqrt{t}} \] \[ x^4=t^2 \] Substitute: \[ I=\int_0^\infty t^2 e^{-t}\cdot \frac{dt}{2\sqrt{t}} \] \[ I=\frac12\int_0^\infty t^{3/2}e^{-t}dt \] Compare with Gamma Function: \[ n-1=\frac32 \] \[ n=\frac52 \] \[ I=\frac12\Gamma\left(\frac52\right) \] \[ \Gamma\left(\frac52\right) = \frac32\cdot\frac12\sqrt{\pi} \] \[ = \frac{3\sqrt{\pi}}{4} \] \[ I=\frac12\times\frac{3\sqrt{\pi}}{4} \]

\[ \boxed{\frac{3\sqrt{\pi}}{8}} \]

Example 2

Evaluate:

\[ I=\int_0^{\infty} x^{7/2}e^{-x}dx \]

Solution

\[ n-1=\frac72 \] \[ n=\frac92 \] \[ I=\Gamma\left(\frac92\right) \] \[ = \frac72\cdot\frac52\cdot\frac32\cdot\frac12\sqrt{\pi} \] \[ = \frac{105\sqrt{\pi}}{16} \]

\[ \boxed{\frac{105\sqrt{\pi}}{16}} \]

Example 3

Evaluate:

\[ I=\int_0^{\infty} e^{-x^2}dx \]

Solution

\[ x^2=t \] \[ dx=\frac{dt}{2\sqrt{t}} \] Substitute: \[ I=\frac12\int_0^\infty t^{-1/2}e^{-t}dt \] \[ n-1=-\frac12 \] \[ n=\frac12 \] \[ I=\frac12\Gamma\left(\frac12\right) \] \[ =\frac12\sqrt{\pi} \]

\[ \boxed{\frac{\sqrt{\pi}}{2}} \]

Example 4

Evaluate:

\[ I=\int_0^{\infty} x^{9/2}e^{-x^2}dx \]

Solution

\[ x^2=t \] \[ dx=\frac{dt}{2\sqrt{t}} \] \[ x^{9/2}=t^{9/4} \] Substitute: \[ I=\frac12\int_0^\infty t^{7/4}e^{-t}dt \] \[ n-1=\frac74 \] \[ n=\frac{11}{4} \] \[ I=\frac12\Gamma\left(\frac{11}{4}\right) \]

\[ \boxed{\frac12\Gamma\left(\frac{11}{4}\right)} \]

Example 5

Evaluate:

\[ I=\int_0^{\infty} y^3e^{-y^2}dy \]

Solution

\[ y^2=t \] \[ 2y\,dy=dt \] \[ dy=\frac{dt}{2\sqrt{t}} \] \[ y^3=t^{3/2} \] Substitute: \[ I=\frac12\int_0^\infty te^{-t}dt \] \[ n-1=1 \] \[ n=2 \] \[ I=\frac12\Gamma(2) \] \[ \Gamma(2)=1 \]

\[ \boxed{\frac12} \]

Beta Function

Beta Function Formula: \[ \beta(m,n)=\int_0^1 x^{m-1}(1-x)^{n-1}dx \]
Relation Between Beta and Gamma Function: \[ \beta(m,n)=\frac{\Gamma(m)\Gamma(n)}{\Gamma(m+n)} \]

Example 1

Evaluate:

\[ I=\int_0^{1/2} x^3\sqrt{1-4x^2}\,dx \]

Solution

\[ 2x=\sin\theta \] \[ x=\frac{\sin\theta}{2} \] \[ dx=\frac{\cos\theta}{2}d\theta \] \[ \sqrt{1-4x^2}=\cos\theta \] Substitute: \[ I=\frac1{16} \int_0^{\pi/2} \sin^3\theta\cos^2\theta\,d\theta \] Using Beta Formula: \[ \int_0^{\pi/2} \sin^{m-1}\theta \cos^{n-1}\theta\,d\theta = \frac12\beta\left(\frac m2,\frac n2\right) \] \[ I=\frac1{16}\cdot\frac12\beta\left(2,\frac32\right) \] \[ I=\frac1{120} \]

\[ \boxed{\frac1{120}} \]

Example 2

Evaluate:

\[ I=\int_0^8 x^{1/3}(8-x)^{2/3}dx \]

Solution

\[ x=8t \] \[ dx=8dt \] Substitute: \[ I= 64 \int_0^1 t^{1/3}(1-t)^{2/3}dt \] Compare with Beta Function: \[ m=\frac43,\quad n=\frac53 \] Thus, \[ I= 64\beta\left(\frac43,\frac53\right) \]

\[ \boxed{ 64\beta\left(\frac43,\frac53\right) } \]

Example 3

Evaluate:

\[ I=\int_0^1 x^{-1/3}(1-x)^{-1/3}dx \]

Solution

\[ m-1=-\frac13 \] \[ m=\frac23 \] \[ n=\frac23 \] Thus, \[ I= \beta\left(\frac23,\frac23\right) \] Using Gamma Relation: \[ I= \frac{ \Gamma(2/3)\Gamma(2/3) }{ \Gamma(4/3) } \] \[ I= \frac{2\pi}{\sqrt3} \]

\[ \boxed{\frac{2\pi}{\sqrt3}} \]

Example 4

Evaluate:

\[ I=\int_0^1 x^3(1-x)^5dx \]

Solution

\[ m=4,\quad n=6 \] \[ I=\beta(4,6) \] \[ I= \frac{ \Gamma(4)\Gamma(6) }{ \Gamma(10) } \] \[ = \frac{ 3!5! }{ 9! } \] \[ = \frac1{504} \]

\[ \boxed{\frac1{504}} \]

Example 5

Evaluate:

\[ I=\int_0^2 x^3\sqrt{2-x}\,dx \]

Solution

\[ x=2t \] \[ dx=2dt \] \[ 2-x=2(1-t) \] \[ \sqrt{2-x}=\sqrt2(1-t)^{1/2} \] Substitute: \[ I= 16\sqrt2 \int_0^1 t^3(1-t)^{1/2}dt \] Compare with Beta Function: \[ m=4,\quad n=\frac32 \] Thus, \[ I= 16\sqrt2 \beta\left(4,\frac32\right) \]

\[ \boxed{ 16\sqrt2\beta\left(4,\frac32\right) } \]
